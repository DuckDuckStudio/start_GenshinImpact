如何修改程序的灵敏度？

在第18行 return white_ratio > 0.9 中可以修改，默认0.9，既白色区域超过整个屏幕的90%时，将会执行原始启动，可以自行修改
----------------------------------------------------------------------------------------------------------------
程序运行时CPU占用太高了怎么办？

在程序执行中，默认不会进行限制，如果程序运行后，CPU实在吃不消，在第65行 time.sleep(0) 中进行修改，其中的数字0可以修改为0.5，即为每0.5秒检测一次，可根据需要修改
